# data-management

> csdn博客每日阅读量和访问量统计的前后端,数据来源依赖爬虫(https://github.com/guosimin/python-spider/tree/master/csdnTest/csdn-test.py)

## Build Setup

1.下载代码，并启动并构建前端项目     
```
# install dependencies

npm install

# serve with hot reload at localhost:8080

npm run dev
```
2.启用后端   

```
node app.js
```
3.启动完成后访问：http://localhost:8081/#/charts
